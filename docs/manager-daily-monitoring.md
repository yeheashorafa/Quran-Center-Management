# متابعة الحلقات حسب التاريخ

توفر لوحة المدير تبويباً لمتابعة الحلقات المطلوبة في تاريخ محدد.

## القواعد

- اليوم يستخرج من التاريخ ولا يرسله المستخدم كقيمة مستقلة.
- تظهر الحلقة فقط عندما يوجد لها جدول فعّال يطابق يوم التاريخ المحدد.
- عدد الطلاب المتوقع يحسب من التسجيلات السارية في التاريخ نفسه.
- الحلقة تعد «لم تسجّل» عندما لا توجد سجلات طلاب فعلية، حتى لو تم إنشاء جلسة فارغة.
- الجلسة المكتملة أو المقفلة تحسب ضمن الجلسات المكتملة.
- جميع الإحصائيات تقرأ من `memorization_sessions` و`session_record_items` و`session_activities`.

## الصلاحية

مسار API محمي بصلاحية:

```text
sessions.read.all
```

## المسار

```text
GET /api/manager/monitoring?date=YYYY-MM-DD
```
